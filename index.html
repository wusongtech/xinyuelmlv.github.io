Git bash 是以命令行的方式管理github项目，为了明确是谁进行了相应的操作，在使用git管理github前需要进行git的基本信息初始化，在git bash页面执行以下命令：

1.初始化基本信息

 git config --global  user.name  'xinyue'

 git config  --global  user.email  '958674218@qq.com'

2.初始化一个新的Git仓库

   a. 新建一个文件夹

   b. 在gitbash中以命令行的模式进入新建的文件夹，执行命令git init，完成git仓库的初始化。

 

另外，执行 git config --list可以查看当前git的初始配置
